# Advisory corpus (LLMs)

**Read `BASE.md` first** — stable DAO orientation, ledger map, and canonical URLs. Then open `index.json` and the latest dated file under `snapshots/`.

Daily evidence digests are generated by `market_research/scripts/generate_advisory_snapshot.py` (git + `CONTEXT_UPDATES` + Beer Hall excerpts; optional `--with-sheet-sales`).

- `BASE.md` — slow-changing strategic context (not regenerated by the snapshot script).
- `snapshots/YYYY-MM-DD.md` — one file per UTC day (last write wins if re-run same day).
- `index.json` — schema v2+: `base_markdown`, `read_order`, `canonical_context_urls`, `snapshots`.
- Browse: https://github.com/TrueSightDAO/ecosystem_change_logs/tree/main/advisory
