# Advisory snapshots

LLM-oriented **evidence digests** (git + context files + Beer Hall excerpts), generated by `market_research/scripts/generate_advisory_snapshot.py`.

- `snapshots/YYYY-MM-DD.md` — one file per UTC day (last write wins if re-run same day).
- `index.json` — newest-first list with paths relative to repo root.
- Raw GitHub: https://github.com/TrueSightDAO/ecosystem_change_logs/tree/main/advisory
